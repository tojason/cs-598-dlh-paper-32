# cs-598-dlh-paper-32

# Dependencies
* Python >= 3.8.11
* Please use conda to install all the dependencies using the command
  * `conda env create -f environment.yml`

# How to run the code
Once you install the dependency, please run the following code to open Jupyter Notebook on your terminal (Mac OS/Linux system)
* `jupyter notebook`

# Data download instruction
The filtered data of MIMIC-III is dataset/t1 folder. This subset of MIMIC-III is already decoded that does not contains any original information from the original MIMIC-III.
